Preguntas y Respuestas sobre TypeScript

¿Qué es TypeScript?

Es un superset de JavaScript que añade tipado estático.

¿Qué ventajas ofrece TypeScript frente a JavaScript puro?

Detección de errores en tiempo de desarrollo, mejor autocompletado y mantenimiento.

¿Cómo se instala TypeScript en VSCode?

Ejecuta npm install -g typescript en la terminal.

¿Cómo se compila un archivo TypeScript para convertirlo a JavaScript?

Usa tsc archivo.ts.

¿Qué extensión de archivo se utiliza para los archivos TypeScript?

.ts

¿Cómo se define un tipo de dato en TypeScript?

let edad: number = 25;

¿Qué son los tipos de unión y cómo se declaran?

Permiten múltiples tipos: let valor: string | number;

¿Qué es un tipo literal en TypeScript y para qué se utiliza?

Es un valor específico permitido: let tamaño: 'S' | 'M' | 'L';

¿Qué son los tipos enumerados (Enums) y cómo se crean?

Conjunto de constantes nombradas:
enum Color { Rojo, Verde, Azul }

¿Cómo se define un tipo alias y cuándo es útil?

Con type:
type ID = string | number;
Útil para reutilizar tipos complejos.

¿Cómo se definen las funciones con tipos en TypeScript?

function suma(a: number, b: number): number { return a + b; }

¿Qué son los parámetros opcionales y los parámetros predeterminados en funciones?

Opcional: function saludar(nombre?: string)
Predeterminado: function saludar(nombre: string = 'Usuario')

¿Cuáles son algunas mejores prácticas para escribir código limpio y mantenible en TypeScript?

Usar tipado explícito, evitar any, usar interfaces o tipos, dividir el código en módulos y documentar funciones.
